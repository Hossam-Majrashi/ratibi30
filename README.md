## 📖 About

**ratibi30** is a personal finance application built with Flutter to help users organize salary-based budgeting in a simple and practical way. The app focuses on daily spending control, expense tracking, savings awareness, charity management, and flexible calculation periods.

It is designed for users who want a clean offline-first experience with Arabic and English support, customizable day-based budgeting, and a modern light/dark interface.

## ✨ Features

### 💰 **Salary & Budget Management**
- **Salary Setup**: Add your monthly salary in a simple first-time setup flow.
- **Flexible Day Selection**: Choose 5 days, 10 days, 30 days, 31 days, or enter a custom number of days.
- **Daily Budget Calculation**: Automatically calculates your budget based on salary and selected days.
- **Remaining Balance Tracking**: Shows what is left for today and for the full period.

### 🧾 **Expense Tracking**
- **Add Expenses**: Save expenses with amount, title, category, and date.
- **Expense List**: Review all saved expenses in one place.
- **Category Filtering**: Filter expenses for easier tracking.
- **Date-Based Records**: Organize spending by actual dates.

### 🤲 **Savings & Charity**
- **Savings Monitoring**: Track current savings balance.
- **Charity Fund Screen**: Manage charity from its own dedicated section.
- **Fixed or Percentage Charity**: Set charity as a fixed amount or a percentage of salary.
- **Financial Overview**: Charity is included in the dashboard calculations.

### 🌐 **Language & Theme**
- **Arabic and English Support**: Full bilingual interface.
- **Light / Dark Mode**: Choose day mode or night mode.
- **Live Theme Switching**: Theme changes apply immediately.
- **Improved Dark Mode Cards**: Better visual contrast for card components.

### ⚙️ **Settings & Privacy**
- **Custom Settings**: Change salary, days, currency, language, and preferences.
- **Privacy & Terms**: Dedicated privacy section inside settings.
- **Developer Information**: Shows developer name, GitHub, and email.
- **Offline Storage**: Saves data locally on the device.

## 🚀 Installation

### Prerequisites
- **Flutter SDK** 3.4.0 or higher.
- **Dart SDK** compatible with Flutter.
- Any Flutter-supported editor such as Android Studio or VS Code.

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/HossamMajrashi/ratibi30.git
   cd ratibi30
   ```

2. **Install dependencies**
   ```bash
   flutter pub get
   ```

3. **Run the application**
   ```bash
   flutter run
   ```

## 🎮 Usage

### First Use
1. Open the app.
2. Choose the language.
3. Choose day mode or night mode.
4. Enter the monthly salary.
5. Select the number of days that matches your plan.
6. Save and continue to the dashboard.

### Daily Use
- Add expenses from the **Add** screen.
- Review records from the **Expenses** screen.
- Manage charity from the **Charity Fund** screen.
- Check progress in **Reports** and **Dashboard**.
- Change settings anytime from the **Settings** screen.
